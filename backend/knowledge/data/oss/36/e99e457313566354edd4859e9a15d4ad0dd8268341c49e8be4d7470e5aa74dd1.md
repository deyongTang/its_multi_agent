# 知识库 36

## 标题
如何使用 WP IE 浏览器打开多个页面并互相切换?

## 问题描述
Windows Phone 中集成的 IE 浏览器，也继承了电脑版 IE 7 引进的多标签功能，即可以同时在不同标签中打开多个子页面；除此之外，还可以在不同浏览器之间快速的切换。

## 分类
主类别: 预装软件
子类别: 手机应用

## 关键词
WP, IE, 浏览器, 多个页面

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
1、打开 IE 浏览器

在“**开始屏幕**”或者已安装应用列表中点击 IE 的图标，进入 IE 浏览器：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_751.png)

2、新建子页面

点击右下角“……”按钮，在菜单中选择“**索引标签**”；

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_960.png)

在界面中，点击“**+**”按钮可以新建索引标签；

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_349.png)

新建索引标签后，点击新建的“**新索引标签**”可以进入此子页面，输入网址可以在此子页面中浏览网页。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_780.png)

3、子页面切换

子页面的切换有两种方式：

（1） 按照步骤2中的方式，在“**索引标签**”界面中，选择对应的标签，即可查看此子页面的内容；

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_257.png)

（2） 长按 Windows Phone [硬件](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17962)左向箭头返回按钮，即可查看当前所有打开的应用，包括所有 IE 的子页面；

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_489.png)

长按返回按钮后，即可查看到所有索引标签，在屏幕上左右滑动可以查看所有的界面，点击任意界面即可进入相应的子页面浏览。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344226688754_766.png)
