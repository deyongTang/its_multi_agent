# 知识库 248

## 标题
Windows XP如何启动防火墙

## 问题描述
Windows XP 进入控制面板中，进行启动防火墙操作。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
XP, 查看, 防火墙, 设置, 管理, Windows, 安全

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、按顺序点击“**开始**”--->“**[控制面板](/detail/kd_18077.html)**”，打开“**控制面板**”，如下图：

![如何设置windows xp系统启用防火墙](https://exp-picture.cdn.bcebos.com/c255efc595ee41c14d6fade58d88912ca4ca9be2.jpg?x-bce-process=image%2Fresize%2Cm_lfit%2Cw_500%2Climit_1%2Fformat%2Cf_auto%2Fquality%2Cq_80)

2、点击“**安全中心**”，打开“**Windows 安全中心**”对话框。如下图：

![如何设置windows xp系统启用防火墙](https://exp-picture.cdn.bcebos.com/f0848bee41c1b727571b073e1a2ca5cadde898e2.jpg?x-bce-process=image%2Fresize%2Cm_lfit%2Cw_500%2Climit_1%2Fformat%2Cf_auto%2Fquality%2Cq_80)

3、点击“**Windows防火墙**”，打开“**Windows防火墙**”对话框，选中“**启用（推荐）**”单选按钮，最后单击“**确定**”按钮，完成Windows XP的防火墙设置。如下图：

![如何设置windows xp系统启用防火墙](https://exp-picture.cdn.bcebos.com/94af5fc1b727ac53fac0909a2ecadce8914899e2.jpg?x-bce-process=image%2Fresize%2Cm_lfit%2Cw_500%2Climit_1%2Fformat%2Cf_auto%2Fquality%2Cq_80)
