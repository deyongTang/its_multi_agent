## 标题
Windows 7系统下如何恢复备份数据？

## 问题描述
Windows 7支持文件备份和系统映像备份，前者是个人文件（如文档、音乐、图片和视频）的副本，后者是驱动器的准确映像。Windows 7 恢复备份的步骤：6、选择需要还原的方式，可以恢复备份的所有文件，也可以通过浏览选择其中的文件恢复。联想远程数据恢复服务，不成功不收费！

## 解决方案
Windows 7支持文件备份和系统映像备份，前者是个人文件（如文档、音乐、图片和视频）的副本，后者是驱动器的准确映像。如果希望恢复备份数据，需要先做备份，然后做还原操作。

Windows 7 恢复备份的步骤：  
  
1、单击“**开始**”按钮，单击“**控制面板**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image001.jpg)

2、单击“**系统和安全**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image002.jpg)

3、单击“**备份和还原**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image003.jpg)

4、单击“**选择要从中还原文件的其他备份**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image004.jpg)

5、选择一个备份， 点击“**下一步**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image005.jpg)

6、选择需要还原的方式，可以恢复备份的所有文件，也可以通过浏览选择其中的文件恢复。这里以“**选择此备份中的所有文件**”为例，点击“**下一步**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image006.jpg)

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image007.jpg)

7、选择恢复的位置，此处选择“**在原始位置**”，点击“**还原**”；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image008.jpg)

8、正在进行还原的界面；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image009.jpg)

9、还原完成后，点击“**完成**”即可。

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/125.files/image010.jpg)
