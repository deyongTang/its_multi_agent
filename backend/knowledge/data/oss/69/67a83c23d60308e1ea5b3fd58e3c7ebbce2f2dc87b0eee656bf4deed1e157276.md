## 标题
开机蓝屏或提示 登录进程初始化失败 问题的解决方案（Windows 7）

## 问题描述
Win7系统下开机蓝屏或提示登录进程初始化失败问题的原因分析以及解决方案。

## 分类
主类别: 操作系统故障
子类别: 系统登录问题

## 关键词
登陆进程初始化失败, 登陆失败进不去系统, Windows7

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
**故障现象：**

6月12日，部分用户在启动时出现“蓝屏”或提示“登录进程初始化失败”，无法进入桌面

  Windows 7用户（如下图）。

  Windows Vista用户，请点击[这里](http://support1.lenovo.com.cn/lenovo/wsi/htmls/detail_1371131444165.html)。

  Windows XP用户，请点击[这里](http://support1.lenovo.com.cn/lenovo/wsi/htmls/detail_1371130865828.html)。

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20130613/1371129679703_260.jpg) 

**原因分析：**

初步确认本次部分安装有金山软件的Windows用户出现“登录进程初始化失败”问题与Windows系统安全更新补丁（KB2839229）有关。

 

**解决方案：**

Windows 7系统 

1、在出现上述界面时，按住电源开关10秒强制关机，确认关机后，再次点击开关键开机，并连续点击键盘上的F8键，直到出现高级启动菜单（如下图）。用键盘右下角的方向键“↓”键选择“安全模式”按“回车”；

 ![](https://webdoc.lenovo.com.cn/lenovowsi/20130613/1371129679703_843.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20130613/1371129679703_489.jpg)

2、大约30秒后，系统会提示“配置Windows Update失败，还原更改，请勿关闭计算机”，此时您不需要做任何操作，耐心等待；

 ![](https://webdoc.lenovo.com.cn/lenovowsi/20130613/1371129679703_68.jpg)

3、30秒后电脑会自动重启，再次进入Windows启动界面（此时系统会自动更新配置信息）；

 ![](https://webdoc.lenovo.com.cn/lenovowsi/20130613/1371129679703_594.jpg)

4、30秒后，就可以进入系统桌面了，您已经可以正常使用您的电脑。

5、进入系统后，为确保"登录进程初始化失败"的问题不再次发生，请将您的金山卫士和毒霸等软件，更新到最新版本。
