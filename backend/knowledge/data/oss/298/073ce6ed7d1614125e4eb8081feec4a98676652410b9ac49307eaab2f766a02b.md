# 知识库 298

## 标题
如何在 Windows 系统下安装打印机硬件驱动程序

## 问题描述
在 Windows 系统下安装打印机硬件驱动程序的操作方法。

## 分类
主类别: 接口与外置设备
子类别: 打印机/扫描仪连接

## 关键词
关机, 开机, 运行, 报错, 设置, 管理, Windows, 系统, 打印机, 驱动

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
安装打印机驱动通常有“使用随机光盘安装驱动程序”和“使用下载的软件安装驱动程序”两种方法；两种安装方法类似，本文以“使用随机光盘安装驱动程序”为例：

1、接通一体机电源后，使用 USB 连接线将一体机和电脑连接，稍后电脑屏幕会弹出找到新硬件窗口，请放入随机附带的驱动程序光盘，然后选中“自动安装软件（推荐）”选项，点击“下一步”按钮；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/154.files/image001.jpg)

2、新硬件安装向导会自动从光盘中寻找可以使用的驱动程序，请在搜索完成后点击“下一步”按钮；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/154.files/image003.jpg)

3、接下来系统可能会提示安装的软件没有通过测试，请直接点击“仍然继续”按钮；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/154.files/image005.jpg)

4、接下来新硬件安装向导会开始复制驱动程序的文件，请等待复制完成；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/154.files/image007.jpg)

5、文件复制完成后，安装向导会提示设备安装已经完成，请点击“完成”按钮；

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/154.files/image009.jpg)

6、接下来电脑会提示正在安装新硬件及端口等硬件设备，请等待系统自动设置完成；这样打印机驱动安装完毕。

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/154.files/image011.jpg)
