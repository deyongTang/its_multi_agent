# 知识库 74

## 标题
Windows 7系统下开机蓝屏提示问题的解决方案

## 问题描述
通过系统或安全软件进行补丁更新后，可能会导致用户电脑重启后出现开机0xc0000005,0xc0000008e蓝屏，同时无法开机的现象。此现象与KB2859537补丁有关。

## 分类
主类别: 操作系统故障
子类别: 蓝屏死机

## 关键词
Windows 开机蓝屏, xp开机蓝屏, 蓝屏, 0xc0000005, 0xc0000008e

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
[软件](/detail/kd_17963.html)进行补丁更新后，可能会导致用户电脑[重启](/detail/kd_17977.html)后出现开机0xc0000005,0xc0000008e蓝屏，同时无法开机的现象。此现象与KB2859537补丁有关。

**影响系统：**

XP（SP3）；[vista](/detail/kd_17890.html)\_32/64（SP2）；2003（SP2）；win7\_32/64（SP1）;

临时方案建议在开机进按F8进入[安全模式](/detail/kd_17783.html)，将此补丁卸载，重新[启动](/detail/kd_17514.html)即可。

**原因分析：**

初步确认为部分杀毒软件与Windows系统安全更新补丁（KB2859537）有关。

**解决方案：**

1、在出现上述界面时，按住电源开关10秒强制关机，确认关机后，再次点击开关键开机，并连续点击键盘上的F8键，直到出现高级启动菜单（如下图）。用键盘右下角的方向键“↓”键选择“安全模式”按“回车”。

![](https://webdoc.lenovo.com.cn/lenovowsi/20130815/1376537708996_174.png)

![](https://webdoc.lenovo.com.cn/lenovowsi/20130815/1376537708996_664.jpg)

2、[Windows 7](/detail/kd_17984.html)系统，请点击开始---[控制面板](/detail/kd_18077.html)---程序---程序和功能，点击查看已安装的更新，找到KB2859537补丁，并点击卸载，删除补丁后请重新启动计算机即可。如下图所示：

![](https://webdoc.lenovo.com.cn/lenovowsi/20130815/1376537708996_636.png)

![](https://webdoc.lenovo.com.cn/lenovowsi/20130815/1376537708996_642.png)

3、进入系统后，为确保"蓝屏报错"的问题不再发生，请将杀毒软件更新到最新版本。
