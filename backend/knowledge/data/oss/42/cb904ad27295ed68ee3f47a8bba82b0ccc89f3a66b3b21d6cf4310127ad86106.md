## 标题
如何在 Windows Phone 中查看后台任务?

## 问题描述
在 Windows Phone 中，有一类应用程序具有在后台运行的功能，即使这些应用程序没有被打开，它们也会在后台执行命令，这类程序被称为后台任务。下文将讲述对于怎样查看后台任务。

## 分类
主类别: 预装软件
子类别: 手机应用

Windows Phone, 后台任务, 程序, 进程

## 解决方案
在 Windows Phone 中，有一类应用程序具有在后台运行的功能，即使这些应用程序没有被打开，它们也会在后台执行命令，这类程序被称为后台任务。

常见的后台任务可能执行的操作包括更新主界面 Live Tile，获取实时信息（如：天气预报、新闻等），也可能是获取用户地理位置，更新股市行情等。

这些后台任务在带给用户方便的同时，也可能引起一些其他问题，如电池电力耗损、[网络流量](/detail/kd_17572.html)增加或者泄露用户位置信息等。

通过 Windows Phone 手机的设置功能，可以查看用户手机中有哪些应用程序开启了后台任务功能，并且，用户可以根据自己的需要对其进行具体设置。

具体操作方法如下：

●打开手机中的“**设置（Settings）**”：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344240234065_60.jpg)

●将屏幕向左滑动，切换到“**应用程序**”一栏：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344240234065_927.jpg)

●打开第一项“**后台任务**”设置：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344240234065_502.jpg)

●这里列出的即是当前手机中所有的具有后台任务功能的应用程序，在这里可以看到，一些后台应用是“**开启**”状态的，而另外一些，是“**关闭**”的。

对于开启后台任务的应用程序，用户可以点击程序名称，查看该后台任务的具体说明，并进行关闭操作：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344240234065_338.jpg)

●对于处于关闭状态的后台任务，用户可以在设置中选择“**当应用程序下次[启动](/detail/kd_17514.html)时开启后台任务**”：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344240234065_174.jpg)

这个选项并不意味着该应用程序的后台任务会立即运行，而是等到用户下次手动运行该应用程序时，系统会为其重新开启后台任务功能。
