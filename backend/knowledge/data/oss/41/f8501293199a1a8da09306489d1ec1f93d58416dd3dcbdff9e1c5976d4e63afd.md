# 知识库 41

## 标题
巧用 Windows Phone 搜索功能

## 问题描述
Windows Phone 内置的 Bing 搜索非常强大，您可以在这里搜索到任何您想要的内容，无论是文章、应用、音乐还是临近的餐馆。

## 分类
主类别: 预装软件
子类别: 手机应用

## 关键词
Windows Phone, 搜索功能, 小技巧，搜索

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
[Windows Phone](/detail/kd_17705.html) 内置的 Bing 搜索非常强大，您可以在这里搜索到任何您想要的内容，无论是文章、应用、音乐还是临近的餐馆。

在任何界面下，都可以使用屏幕上的“**搜索**”按钮，打开Bing，除了传统搜索框，Bing 还为用户提供了4种独特的搜索方式：“**临近区域**”可以使用用户当前的地理位置，查找周围的餐饮、娱乐等信息；通过“**音乐**”搜索，用户可以哼唱或者播放一段音乐，Bing 会返回相应的曲目信息；“**视觉**”搜索就是通过摄像头读取对象信息进行搜索，可以扫描的内容包括[二维码](/detail/kd_17908.html)、条形码、微软 Tag 等，也可以是书本的封面，或者，可以通过摄像头扫描文字并进行翻译。也可以使用“**语音**”搜索，说出需要搜索的内容即可，但是 Windows Phone 语音功能暂时不支持中文。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_484.jpg)

“**临近**”搜索点击 Bing 搜索中位于菜单栏最左侧的“**临近**”搜索按钮，您的手机会以您所处的位置为中心，搜索周边的餐饮、娱乐等信息：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_205.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_492.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_736.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_460.jpg)

屏幕最上方显示了您当前所处位置的周边地图，并在地图中标出了搜索结果，地图下方是您所处的位置，最下方是搜索结果，其中包括了餐饮、购物、周边活动和其他地标信息。当您滑动屏幕时，最上方的地图会随之移动到显示对应的结果。

您也可以点击地图将其展开至全屏浏览：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_94.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_339.jpg)

在这里，您还可以进行导航、查看交通状况、切换地图模式、添加收藏等操作。

“**音乐**”搜索

当您听到音乐却不知其曲名时，或者想起一首曲子却想不起名字时，可以使用Bing的“**音乐**”搜索进行查找。

打开Bing搜索的“**音乐**”搜索，此时Windows Phone会聆听由[麦克风](/detail/kd_17501.html)传递进来的音乐：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_475.jpg)

大约经过30秒，Bing 会返回音乐识别的结果：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_449.jpg)

结果包含了专辑名称、音乐家和曲名。您可以通过“**商店**”浏览专辑详细信息：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_849.jpg)

您在这里可以试听曲目，或者购买音乐：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_216.jpg)

在Bing搜索中，您可以通过菜单栏中的“**音乐搜索历史**”按钮，查看曾经识别的音乐记录和相关信息：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_642.jpg)

“**视觉**”搜索

用处最广的当属 Bing 独特的“**视觉**”搜索功能，打开 Bing 的“**视觉**”搜索，并将照相机面对需要查找的对象。

例如，对准条形码或二维码，相机会自动识别条码内容，并显示相关信息：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_841.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_938.jpg)

点击搜索结果，可以查看产品详细信息，包括名称、售价和描述等：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_917.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_349.jpg)

通过下方的“**历史**”按钮，您可以查看到过去扫描过的所有内容：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_688.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_709.jpg)

并且可以再次点击查看详情。

如果您扫描的对象是文字，请按下“**扫描文字**”按钮，此时，Bing 会自动识别出文字内容，并将识别结果标记出来：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_23.jpg)

您只需按下“**翻译**”按钮，即可选择目标语言进行翻译：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_421.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_561.jpg)

您还可以将相机对准图书、CD封面等，进行封面识别，例如这里将扫描的对象换成了图书《Gone with the Wind》封面，Bing 成功识别出了该书并返回了多条搜索结果：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_248.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_356.jpg)

点击搜索结果，可以查看产品信息，或者调用第三方[软件](/detail/kd_17963.html)（如这里的IMDb）获取更多资讯。

“**语音**”搜索

在 Bing 中，可以使用语音代替键盘输入，打开 Bing，选择“**语音**”搜索，说出您需要搜索的关键字（暂时仅支持英语）：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_601.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_517.jpg)

点击“**搜索**”即可，Bing 内置的微软 TellMe 功能会进行语音识别，并返回搜索结果：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_361.jpg)

在搜索结果页面的搜索框中，您依然可以通过点击搜索框中的“**语音**”按钮进行语音搜索。

搜索设置

通过 Bing 的设置，可以对搜索功能进行调整，打开 Bing 搜索菜单中的“**设置**”：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_417.jpg)

在这里，您可以选择是否允许 Bing“**使用您的位置**”信息，或者是否允许“**将位置信息发送给微软 Tags 服务**”，“**允许在屏幕锁定时使用搜索按钮**”会让您更加方便的进行搜索：

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_593.jpg)

“**输入建议**”是指在您输入搜索关键字时，Bing 会自动返回匹配关键字列表，另外，还可以控制是否“**允许微软存储和使用来自视觉搜索的图片**”，允许这一功能会帮助微软改进并提高搜索结果和质量，您不用过多担心，微软不会泄露这些内容，也不会将这些内容用作其他用途。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120806/1344238163060_14.jpg)

在设置的最下方，您可以使用“**删除[历史记录](/detail/kd_17791.html)**”按钮清空 Bing 搜索历史，删除内容包括输入的关键字历史、视觉搜索和音乐搜索历史。
