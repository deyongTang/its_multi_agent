# 知识库 297

## 标题
使用Windows Install Clean Up卸载软件的方法

## 问题描述
介绍使用Windows Install Clean Up卸载软件的方法，操作步骤。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
Windows Install Clean Up, 卸载

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、软件下载，在百度搜索“**Windows Install Clean Up**”，会搜索到许多下载链接，下载完成后，以管理员身份运行；

2、屏幕弹出 “**Windows Install Clean Up Setup**” 安装向导，点击“**Next**”；

3、选择“**I Accept the License Agreement**”，点击“**Next**” ；

4、点击“**Next**”继续；

5、点击“**Finish**”,安装完毕；

6、启动Windows Install Clean Up，选择需要卸载软件的名称，点击“**Remove**”按钮；

![](https://img0.pconline.com.cn/pconline/2212/09/15694524_20210113115805_31642_thumb.jpg)

7、Windows Install Clean Up提示，选择的应用程序将被卸载，点击“**确定**”；

8、应用程序卸载完毕。
