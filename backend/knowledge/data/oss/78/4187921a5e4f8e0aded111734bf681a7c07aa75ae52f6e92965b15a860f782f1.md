## 标题
联想G470笔记本亮度无法调节的解决方案

## 问题描述
本文提供了针对联想Lenovo G470笔记本电脑亮度无法调节的问题的解决方案。首先，用户可以尝试使用Fn键结合方向键来调整屏幕亮度。如果这种方法无效，对于Windows 7或Windows 8系统，可以通过控制面板中的电源选项来调节亮度。此外，用户应检查是否安装了AMD Catalyst Install Manager程序，如果存在，建议卸载后重新下载并安装适合用户系统的显卡驱动。如果问题依旧无法解决，建议联系当地联想服务中心或使用联想专家一对一应用指导服务。

## 解决方案
联想 G470机型推荐安装Windows 7或Windows 8操作系统。如果您安装了Windows XP系统，官方驱动对新机型仅提供基础支持，无法保证XP系统下所有功能都可以使用。

联想 G470笔记本亮度调节可使用以下两种方法。如果都无效，请联系当地[联想服务中心](https://newsupport.lenovo.com.cn/serverNet.html)。

**方法一：**

在电脑开机进入系统后，先按下键盘左下角的FN”按键不放，再按下上方向键“↑”（提升屏幕亮度）或下方向键“↓”（降低屏幕亮度）即可。键盘按键位置如下图所示：

![](https://webdoc.lenovo.com.cn/lenovowsi/20131010/1381372547171_116.png)

**方法二：**

Win7或Win8系统下，打开计算机---控制面板---查看方式（大图标）---电源选项，在打开的窗口底部屏幕亮度”可见调节滑动条，左右滑动即可调节屏幕亮度。（Win8为例）：

![](https://webdoc.lenovo.com.cn/lenovowsi/20131010/1381372547171_730.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20131010/1381372547171_844.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20131010/1381372547171_941.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/20131010/1381372547171_620.jpg)

如您发现按照上述两种方法调节屏幕亮度无效，请打开控制面板---程序---卸载程序，在列出的程序列表中，检查是否有AMD Catalyst Install Manager”程序选项，如下图所示：

![](https://webdoc.lenovo.com.cn/lenovowsi/20131010/1381372547171_717.jpg)

**如果有此程序，**请在此程序上单击鼠标右键，选择卸载”，按照提示向导卸载完成，并重启电脑；然后根据您的系统类型，[**登录联想官方服务网站**](https://newsupport.lenovo.com.cn)，下载显卡驱动并安装。
