# 知识库 689

## 标题
联想扫描仪1200CU XP下安装步骤，对应网上下载的驱动

## 问题描述
联想扫描仪1200CU XP下下载和安装的步骤

## 关键词
联想扫描仪, 驱动下载, 1200CU, XP

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
下载的[驱动](/detail/kd_17488.html)程序解压缩后有两个文件。

  

运行 XPDriver

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/EAQsH2h796p0LmW8.JPG)

  

选择解压缩路径，然后点击 Un[zip](/detail/kd_17767.html)

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/4EOjCh20R90hr3A7.JPG)

  

解压完之后，运行b1200cuv13

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/tDufNdwE7xSzqM79.JPG)

  

选择安装语言

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/8zOW0BSVvWPb31Q3.JPG)

  

安装完成之后，插入扫描仪电缆线。

  

选择 从列表或指定位置安装（高级）

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/zm6glBFIx6p89C2H.JPG)

  

选择 “在搜索中包括这个位置”，然后点击右侧的 浏览 ，然后将路径指向 C:\\WINDOWS\\TEMP，点击确定，下一步。

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/e0eiN3hAzUe004L4.JPG)

  

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-08-23/7XCM6ndC2074PK1V.JPG)

  

找到驱动程序并复制文件，完成。
