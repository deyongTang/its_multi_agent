# 知识库 274

## 标题
如何在Windows 7输入法之间切换

## 问题描述
在Windows 7输入法之间切换方法的指导。

## 分类
主类别: 操作系统故障
子类别: 输入法问题

## 关键词
Windows 7, 设置, 管理, Windows, 防火墙, 安全, 杀毒软件, 输入法, 切换

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、点击任务栏输入法图标，在弹出的输入法菜单中点击需要的输入法完成切换；

2、如果想用键盘来着切换，请点击开始菜单中的“**控制面板**”；

3、点击“**时钟、语言和区域**”；

4、点击“**区域和语言选项**”；

5、在弹出的对话框中点选“**键盘和语言**”，点击“**更改键盘**”；

6、依照下图所示顺序点击“**高级键设置**”、“**在输入语言之间**”、“**更改按键顺序**”；

7、在弹出的“**更改按键顺序**”对话框中根据自己的使用习惯设置，点“**确定**”完成。
