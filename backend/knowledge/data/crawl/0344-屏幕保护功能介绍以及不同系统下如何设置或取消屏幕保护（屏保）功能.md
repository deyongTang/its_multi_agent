# 知识库 344

## 标题
屏幕保护功能介绍以及不同系统下如何设置或取消屏幕保护（屏保）功能

## 问题描述
本文详细介绍了Windows系统屏幕保护功能的历史、用途以及如何在不同版本的Windows系统中设置屏幕保护程序。

## 分类
问题类别: 屏幕保护程序

## 关键词
屏幕保护, 屏保, 取消屏保, 设置屏保, 设置屏幕保护

## 元信息
创建时间:2024-12-15|版本:3.0

## 解决方案
        Windows系统的屏幕保护功能（Screen Saver），最初设计用来保护CRT显示器（阴极射线管显示器）不受损害。当CRT显示器长时间显示同一图像时，可能会造成像素烧录，即屏幕上的某些像素因为长时间显示同一颜色而损坏。屏幕保护程序通过显示动态图像或静止图像来避免这种情况发生。随着技术的发展，现代显示器（如LCD和LED）不再需要这种保护，但屏幕保护程序仍然被保留作为一项有用的功能，具有多种用途：

1. **防止像素烧录**：虽然现代显示器不容易出现像素烧录，但长时间显示相同图像仍有可能对显示器造成一定影响。
2. **提供视觉享受**：屏幕保护程序可以展示动态图像、照片幻灯片或艺术作品，为用户提供视觉享受。
3. **保护隐私**：当屏幕保护程序启动后，用户必须输入密码才能恢复工作，这有助于保护未保存的工作和个人隐私。
4. **节能**：某些屏幕保护程序可以设置在一定时间无操作后降低显示器亮度或关闭显示器，从而节省能源。
5. **展示信息**：一些屏幕保护程序可以显示时间、日期、名言警句或股票信息等。
6. **延长显示器寿命**：虽然现代显示器不需要屏幕保护程序来防止像素烧录，但它们仍然可以减少显示器的磨损。

Win10系统设置屏幕保护的方法
----------------

1、在 Windows 10系统桌面空白处单击鼠标右键，弹出菜单选择“个性化”，如图：

![win10个性化](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150710142610176.jpg)

2、左侧列表中定位到“锁屏界面”，右边窗口找到并选择“屏幕保护程序设置”，如图：

![win10屏幕保护程序设置](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150710142728696.jpg)

3、在“屏幕保护程序设置”窗口，点击下拉的小箭头，选择不同的屏保程序，然后点击“确定”即可，如图：

![win10屏保](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150710142801591.jpg)

Win8系统设置屏幕保护的方法
---------------

1、鼠标右键单击桌面空白处，选择"个性化"；  
  
![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20120604210841114.png)  
  
2、鼠标左键单击选择窗口右下方的"屏幕保护程序"；  
![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20120604210934591.jpg)  
  
3、根据个人喜好在窗口中央位置选择所需要的屏幕保护程序类型，然后更改等待时间；（注：等待时间含义为未操作计算机多长时间后自动进入屏幕保护状态）  
  
![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20120604211000631.jpg)  
  
4、点击窗口最下方"确定"即可。

Win7系统设置屏幕保护的方法
---------------

1、在开始菜单搜索输入框中输入“**屏幕保护程序**”，系统会自动匹配几个选项，点击“**更改屏幕保护程序**”（图示中的三个选项都可以打开“**屏幕保护程序**”）：

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/28.files/image004.jpg)

2、在弹出的“**屏幕保护程序**”对话框“**屏幕保护程序**”选项卡中，屏幕保护程序框内设置屏幕保护程序，可以通过下拉菜单选择屏保的画面，点击“**预览**”按钮预览屏保画面，更改等待的时间，以及选择是否在恢复时显示登录屏幕,设置后点击“**应用**”-“**确定**”按钮开启屏幕保护程序。

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/28.files/image005.jpg)

3、如果想取消屏幕保护功能，请在下拉菜单中点选“**无**”，点击“**应用**”-“**确定**”即可。

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/28.files/image006.jpg)

**Windows XP**系统设置屏幕保护的方法
-------------------------

1、桌面空白区域单击鼠标右键，在弹出的菜单中点击**属性**”：

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/28.files/image001.jpg)

2、在弹出的“**显示属性**”对话框“**屏幕保护程序**”[选项卡](/detail/kd_17796.html)中，屏幕保护程序框内设置屏幕保护程序，可以通过下拉菜单选择屏保的画面，点击**预览**”按钮预览屏保画面，更改等待的时间，以及选择是否在恢复时使用密码保护（此功能需要在“**[控制面板](/detail/kd_18077.html)**”中“**用户帐户**”下设置用户密码，否则不起作用）,设置后顺序点击“**应用**”-“**确定**”按钮开启屏幕保护程序。

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/28.files/image002.jpg)

3、如果想取消屏幕保护功能，请在下拉菜单中点选“**无**”，点击“**应用**”-“**确定**”即可。

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/28.files/image003.jpg)
