**使用微软cleanup\_tool清除Microsoft .NET Framework**

**1、****[点击此处](http://blogs.msdn.com/cfs-file.ashx/__key/CommunityServer-Components-PostAttachments/00-08-90-44-93/dotnetfx_5F00_cleanup_5F00_tool.zip)**打开链接后点击**“保存**”；

**2、**双击压缩包中的"**cleanup\_tool**”；

**3、**若弹出如下提示请点击"**是**”；

**4、**弹出确认窗口后，请单击"**是**”继续；

**5、**弹出协议告示窗口后，按一下"**Yes**”按钮确认并继续；

**6、**然后单击"**Cleanup Now**”（立即清理）继续；

**7、**出现"**Product cleanup succeeded!**”信息后，就表示清理已经完成了。按一下"**Exit**”（退出）关闭窗口；

**8、**最后，重启电脑以后可以重新安装您所需要的.NET Framework。
