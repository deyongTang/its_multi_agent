# 知识库 952

## 标题
LJ7500激光打印机通过面板设置IP地址的方法

## 问题描述
LJ7500激光打印机通过面板设置IP地址。

## 分类
主类别: 接口与外置设备
子类别: 打印机/扫描仪连接

## 关键词
LJ7500, IP, 设置, 网卡, 网络

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
LJ7500激光[打印机](/detail/kd_18076.html)通过面板设置[IP](/detail/kd_17443.html)地址的方法：

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-03-30/7r9aUPW8ajVLiJab.JPG)  
  
![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-03-30/2M77y63M102T0cKU.JPG)

LJ7500在设置好相应的IP地址、[子网掩码](/detail/kd_17531.html)和[网关](/detail/kd_17369.html)后，必须关闭打印机电源15-30秒后，再重新打开打印机电源（即[重启](/detail/kd_17977.html)打印机），则新设置的IP地址、子网掩码和网关方可生效，此时打印出的配置页上网络设置已更新，然后通过IP地址使用WebTools即可查找到LJ7500打印机。
