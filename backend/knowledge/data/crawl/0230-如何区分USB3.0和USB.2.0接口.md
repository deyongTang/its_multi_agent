# 知识库 230

## 标题
如何区分USB3.0和USB.2.0接口

## 问题描述
本文为您讲述如何根据外观区分USB3.0和USB.2.0接口。

## 分类
主类别: 接口与外置设备
子类别: USB接口相关

## 关键词
USB, USB3.0, USB.2.0, 接口

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
目前电脑的[USB](/detail/kd_17416.html)接口有两种：[USB3.0](/detail/kd_17417.html)和USB2.0。USB3.0接口和USB2.0接口最大的外观区别是，USB3.0是9个针脚，USB2.0是4个针脚。通常情况下USB3.0是蓝色接口，USB2.0是黑色接口。
