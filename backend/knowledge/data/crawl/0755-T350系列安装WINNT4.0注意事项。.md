# 知识库 755

## 标题
T350系列安装WINNT4.0注意事项。

## 问题描述
万全T350系列安装WINNT4.0操作系统时， 需注意以下三点。

## 分类
主类别: 操作系统故障
子类别: 系统安装与升级

## 关键词
万全, 350, WINNT4.0, BIOS

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
T350 3005、TR350 3013、TR350 3023分别使用的是Intel（INTEL 7500、7501）和Supermicro主板，在安装WINNT4.0操作系统时， 均需注意以下几点：

1、安装之前需要检查BIOS中的设置，将“NT4 Installation Workaround”设置为Enable，安装完毕，并安装SP6补丁后，将“NT4 Installation Workaround”设置为Disable。

2、如果在安装WINDOWS NT4之前，硬盘上安装过其他操作系统，那么要使用MS-DOS的Fdisk /mbr命令重新创建MS-DOS格式的硬盘主引导记录，避免安装失败。

3、因为NT 4.0不支持Hyper-threading技术，建议将BIOS中Hyper-threading项设定为Disable。
