# 知识库 601

## 标题
如何在Windows XP SP2中把拨号连接的防火墙去除？

## 问题描述
如何在Windows XP SP2中把拨号连接的防火墙去除的操作方法指导。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
软件, 功能, 技巧, Windows, 硬件, 防火墙

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
点击“开始”－“所有程序”中的连接，选择当前拨号连接名称，右键点击选择“属性”，在属性对话框中选择“高级”标签页，其中有一个选项“通过限制或阻止来自Internet对此计算机的访问来保护我的计算机和网络”，点击“设置”，

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-06-22/pH1SX7Dh5f83LV3y.JPG)  
  
在弹出的“Windows 防火墙”对话框中选择“关闭”，然后点击确定。

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-06-22/PJwG8H5KokHhdysC.JPG)

这样重新拨号就把该拨号连接的防火墙功能去除了。
