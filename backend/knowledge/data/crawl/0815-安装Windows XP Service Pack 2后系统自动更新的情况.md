# 知识库 815

## 标题
安装Windows XP Service Pack 2后系统自动更新的情况

## 问题描述
安装Windows XP Service Pack 2后系统自动更新的情况

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
自动更新, Windows XP, SP2, windows update

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
自动更新

为了展现[sp2](/detail/kd_17484.html)的全部功能 使用了[windows](/detail/kd_17391.html) [update](/detail/kd_17771.html) 。

首先的区别就是升级的update程序 由v4更改为v5。

![1.gif](/upload/image/20160305/1457177915312260.gif "1457177915312260.gif")

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-12-30/tn8qWfGHDx01V8Qi.gif)

auto update失败为什么呢？

因为我没有注意到这个东西。

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-12-30/QXYUIa2NmE92oiVQ.gif)  
  
如果不选择这里，是没有办法自动更新的。这个功能是和3721很像的。  
  
![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-12-30/0BdKYaH6WO6icOGL.gif)

这就是对我的刚才操作的提示。

选择了安装active[控件](/detail/kd_17778.html)以后，首先就会更新windows update程序。

![3.gif](/upload/image/20160305/1457178004306682.gif "1457178004306682.gif")

和以前的那个不太一样了，这里选择当然只能是安装。

然后系统就会出现以下东西。

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-12-30/tLH9h03eWpnG3lOJ.gif)

界面比以前漂亮多了；

以后的步骤就是和以前一样了 先接受[协议](/detail/kd_17810.html) 然后自动安装，使用推荐就可以；

第一次使用auto update会要求下在两份协议 协议内容我还没有看；

大意就是微软给自己推卸责任的，以后补充。
