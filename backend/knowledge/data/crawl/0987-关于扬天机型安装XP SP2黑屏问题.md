# 知识库 987

## 标题
关于扬天机型安装XP SP2黑屏问题

## 问题描述
扬天机型安装XP SP2黑屏问题的解决方法。

## 分类
主类别: 操作系统故障
子类别: 系统安装与升级

## 关键词
补丁, 驱动, sp2, 黑屏

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
 **故障描述：**

[扬天](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17972)机型（5200[显卡](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17348)配合INTEL E0[CPU](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17345)）在安装XP SP2操作系统过程中黑屏的现象。

**原因分析：**

  

这个问题的原因是由于在安装WINDOWS XP SP2操作系统时，[安装程序](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17506)会读取显卡信息，但由于WINDOWS XP SP2操作系统中保存的显卡信息过旧，与系统中安装的显卡不匹配，从而出现此种现象。即此类现象不是[硬件](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17962)故障，且出现的机率非常低

**解决方案：**

**方法1：**

关机，然后重新[启动](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17514)计算机。系统会在原有的基础上继续安装，安装完操作系统后，按随机提供的[驱动](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17488)安装说明页安装联想驱光盘中的显卡驱动（联想驱动光盘中的5200显卡驱动为最新版本的驱动）。

**方法2：**

    

不使用带SP2补丁的WindowsXP安装盘安装WindowsXP操作系统，在安装完毕后，再打上SP2补丁。

 

另外，Microsoft会与Nvidia共同在 WINDOWS XP 升级版本中升级此类显卡信息。
