# 知识库 346

## 标题
如何建立宽带拨号连接

## 问题描述
本文介绍了建立宽带拨号连接的方法，包含win10，win8，win7和xp系统。

## 分类
主类别: 网络问题
子类别: 宽带上网

## 关键词
建立宽带拨号连接, 宽带拨号, 连接, 拨号

## 元信息
创建时间:2025-09-10|版本:3.0

## 解决方案
Windows 10下如何建立宽带拨号连接
---------------------

要创建拨号链接位置首先要进入到该界面。

进入该界面有3种方式。

1、桌面右下角网络图标，右键打开网络和共享中心；

![打开网络和共享中心](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414215940001.jpg)

2、打开控制面板---选择网络和internet---再选择网络和共享中心；

![打开网络和共享中心](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414215941002.jpg)

![打开网络和共享中心](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414215942003.jpg)

3、 进入桌面，网络图标点击鼠标右键---属性---进入网络和共享中心；

![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220003001.jpg)

那么我们开始设置拨号链接；

在网络和共享中心界面中，我们可以看到窗口中间位置有个设置新的链接和网络；

![设置拨号链接](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220004002.jpg)

点击打开设置新的链接和网络后，我们选中第一个选项连接到internet---下一步；

![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220032001.jpg)

再选择第二个选项，设置新连接；

![设置新链接](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220032002.jpg)

再选择宽带（pppoe）方式；

![宽带（pppoe）方式](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220239001.jpg)

我们需要输入正确的宽带账号和密码，直接点击右下角连接等待连接成功或者我们暂时不输入宽带账号和密码，直接选择跳过，再关闭。

![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220239002.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220240003.jpg)

![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220241004.jpg)

这时我们就可以再回到网络和共享中心窗口中左边，点击更改[适配器](https://iknow.lenovo.com.cn/detail/kd_17364.html)设置；

![更改适配器设置](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220306001.jpg)

打开网络连接窗口，这时我们就可以看到宽带连接图标，对宽带连接图标右键，选择创建快捷方式，弹出提示窗口，选择是，这样我们桌面就可以有一个宽带连接的拨号工具了。

![](https://webdoc.lenovo.com.cn/lenovowsi/new_cskb/uploadfile/20150414220306002.jpg)

Windows 8下如何建立宽带拨号连接
--------------------

1、按Win键切换到传统的桌面界面，在任务栏上找到网络图标，右键任务栏网络图标，选择打开网络和共享中心；

![win8宽度链接-1.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/0c051f867f838bf451d2629a08138e35-1757496562171.jpg&name=win8宽度链接-1.jpg)

2、点击设置新的连接或网络；

![win8宽度链接-2.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/953cccc9c79a3af41c9e88a9f1edb1ba-1757496576536.jpg&name=win8宽度链接-2.jpg)

3、点击连接到 Internet；

![win8宽度链接-3.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/0f3c61183beb7b3a867e31ca231f5556-1757496596850.jpg&name=win8宽度链接-3.jpg)

4、选择宽带 PPPoE；

![win8宽度链接-4.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/25f3a9078bc664f037170ba637217d39-1757496614647.jpg&name=win8宽度链接-4.jpg)

5、输入用户名、密码，点击连接；

![win8宽度链接-5.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/1cdd67bb314f4130d593dd43a76ef661-1757496634382.jpg&name=win8宽度链接-5.jpg)

6、在网络和共享中心，点击更改适配器设置；

![win8宽度链接-6.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/c64e2afb54617ce15265eeda2f8e87f2-1757496652551.jpg&name=win8宽度链接-6.jpg)

7、右键点击宽带连接，选择创建快捷方式；

![win8宽度链接-7.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/0a3121996966398e978aa113b5441ba5-1757496666404.jpg&name=win8宽度链接-7.jpg)

8、选择是，桌面上会出现宽带连接图标。

![win8宽度链接-8.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/8cfd3d999ca7623ba5414d257eeb3ef3-1757496678454.jpg&name=win8宽度链接-8.jpg)

Windows 7下如何建立宽带拨号连接
--------------------

1、选择任务栏右下角“网络和共享中心”，打开后选择“设置新的连接或网络”；

![win7宽度链接.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/fdbf6c3a60b90a0c424f0283727a1820-1757496198368.jpg&name=win7宽度链接.jpg)

2、选择“连接到Internet”，“宽带PPPOE”连接；

![win7宽度链接-1.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/13e880239afda7b57acfd19a524b2ed7-1757496220541.jpg&name=win7宽度链接-1.jpg)

![win7宽度链接-2.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/88b1d624ccdf6f3150764595000d596c-1757496231670.jpg&name=win7宽度链接-2.jpg)

3、在此输入申请的宽带用户名和密码后，选择连接即可拨号成功。

![win7宽度链接-3.jpg](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/99e064aded7324fa83e1c6987ad51c02-1757496245318.jpg&name=win7宽度链接-3.jpg)

Windows XP下如何建立宽带拨号连接
---------------------

1、桌面右键“网上邻居”，选择“属性”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204775150.jpg)

2、选择“创建一个新的连接”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204972983.jpg)

3、按照提示向导选择下一步；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204144212.jpg)

4、选择“连接到Internet”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204713210.jpg)

5、选择“手动设置我的连接”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204775678.jpg)

6、选择“用要求用户名和密码的宽带连接来连接”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204905140.jpg)

7、输入ISP名称，申请的用户名和密码后，设置完成。同时可以选择在桌面建立一个快捷方式。

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160317/1458199204605238.jpg)
