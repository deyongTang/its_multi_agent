# 知识库 343

## 标题
如何设置或禁用Windows Media Player播放时允许运行屏幕保护程序

## 问题描述
在播放视频的时候如果长时间不操作机器，机器会自动运行屏幕保护程序，影响使用感受，下面将指导如何设置或禁用Windows Media Player播放时允许运行屏幕保护程序。

## 分类
主类别: 操作系统故障
子类别: 屏幕保护程序

## 关键词
软件, 功能, 技巧, Windows, 视频, 在线, 屏幕保护, Windows Media Player

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
以Windows Media Player 12为例,其他版本的设置大同小异。

1、打开Windows Media Player，点击菜单栏的“**组织**”，选择“**选项**”，如图：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/f8e23836bda1acfc92e645ad4e25eb28-1714293082665.png&name=mceclip0.png)

2、选择“**播放机**”[选项卡](/detail/kd_17796.html)，在“**播放机设置**”下面可以看到有一个选项“**播放时允许运行屏幕保护程序**”，如果允许运行，选中前面的复选框，如果不允许运行，清除前面的复选框，然后点击“**确定**”退出即可。如图：
