# 知识库 3

## 标题
Vista系统下随机软件的安装与卸载

## 问题描述
None

## 分类
主类别: 预装软件
子类别: 其他随机软件

## 关键词
None

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
  

第一步：[随机](/detail/kd_17344.html)[软件](/detail/kd_17963.html)的安装。打开联想应用，安装随机软件。

  

点击桌面左下角开始——>所有程序——>联想应用——>[驱动](/detail/kd_17488.html)与软件安装——>随机软件安装——>选择需要安装的程序(勾选即可)，点击安装选定软件，即可安装随机软件，请耐心等待，选择确定，[重启](/detail/kd_17977.html)电脑后即可生效。

  

第二步：随机软件的卸载。打开[控制面板](/detail/kd_18077.html)，卸载随机软件。

  

点击桌面左下角开始——>控制面板——>程序和功能——>单击需要卸载的随机软件选择上方的“卸载”，选择“是”完全除去所选应用程序及其所有功能，点击完成，卸载结束。
