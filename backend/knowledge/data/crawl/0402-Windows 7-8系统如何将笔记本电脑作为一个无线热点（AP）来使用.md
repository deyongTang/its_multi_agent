# 知识库 402

## 标题
Windows 7/8系统如何将笔记本电脑作为一个无线热点（AP）来使用

## 问题描述
本文介绍了Win7 或 Win8系统下将笔记本电脑作为一个无线热点（AP）来使用的设置方法。

## 分类
主类别: 网络问题
子类别: 共享上网

## 关键词
win7, win8, 无线热点, win7共享无线

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
**方法一:**  
  
1、[控制面板](/detail/kd_18077.html)---网络和[共享](/detail/kd_17685.html)中心---管理无线网络---添加---创建临时网络。

2、出现**创建临时网络**”的一些说明，点击**下一步**”。

3、在**为您的网络命名并选择安全选项**”中填写各项设置。**保存这个网络**”选项作用为：[重启](/detail/kd_17977.html)电脑或下次无需重新设置（在管理无线网络中保存无线[配置文件](/detail/kd_17761.html)）。

4、在**正在设置网络**”界面等待即可。

5、在**网络已经可以使用**”界面会显示网络热点名称以及密码（显示明文字符）-点击**关闭**”即可。

6、在系统系统通知区域点击网络图标，会显示已建立完毕的热点名称，且为**等待用户**”状态，在其他无线设备中已经可以搜索到此热点（此方法可支持多台设备同时连接）

**方法二:**  
  
1、打开**开始**”---在**搜索**”处输入**[cmd](/detail/kd_17642.html)**”---右击搜索结果---**[以管理员身份运行](/detail/kd_17421.html)**”---如出现**[用户账户控制](/detail/kd_17772.html)**”点击**是**”或**允许**”。如图：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/43e025323c7200ff689ffefca34c1ec3-1714361499622.png&name=mceclip0.png)

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/c0207286f5c32ff5117054a33fcc2ffa-1714361514051.png&name=mceclip1.png)

2、出现**[命令提示符](/detail/kd_17641.html)**”窗口，输入：**netsh wlan set [host](/detail/kd_17528.html)ednetwork mode=allow ssid=aPP-WiFi key=LenovoCC**”   
  
命令有三个参数：  
  
**mode：**是否启用虚拟WiFi[网卡](/detail/kd_17366.html)，改为disallow则为禁用。   
  
**ssid：**无线网名称，最好用英文（以App-WiFi为例）。   
  
**key：**无线网密码，八个以上字符（以LenovoCC为例）。   
  
三个参数可以单独使用，例如只使用mode=disallow可以直接禁用虚拟Wifi网卡。

3、打开**网络和共享中心**”---**更改[适配器](/detail/kd_17364.html)设置**”多了一项，如果有多出的这一项**Microsoft Virtual WiFi Miniport Adapter”**，那么说明前边的设置是正确的。

4、继续在命令提示符中运行：netsh wlan start hostednetwork ，参数中**star”**更换为**stop**”即为关闭无线热点；可以新建一个记事本文件复制：netsh wlan start hostednetwork 然后另存为：[启动](/detail/kd_17514.html)[WIFI](/detail/kd_17433.html)热点.[bat](/detail/kd_17755.html) ；下次开机的时候只需要以**管理员身份运行**”就启动了。   
  
至此，虚拟WiFi的红叉叉消失，WiFi基站已组建好，主机设置完毕。无线设备可以通过搜索连接所设置的无线热点名称即可连接；网络名称和密码可以按照自己喜欢的写。

**方法三：**

1、**网络与共享中心**”---**更改适配器设置**”，进入网络连接界面---拔掉连接到笔记本上的[有线网卡](/detail/kd_18120.html)网线---按下Ctrl，同时选中 **[本地连接](/detail/kd_17639.html)**”和**无线网络连接**” ---右键单击其中任意一个选择[桥接](/detail/kd_18058.html)---等待出现网桥图标以后，插上网线，等待桥接完毕，即可出现一个**网桥**”图标。

2、**设置Internet连接共享：**  
  
在**网络连接**”窗口中，右键单击已连接到Internet的网络连接，选择**属性**”---**共享**”，勾上**允许其他网络用户通过此机算计的Internet连接来连接（N）**”并选择**无线网络连接2**”。

3、确定之后，提供共享的网卡图标旁会出现**共享的**”字样，表示**宽带连接**”已共享至**无线连接2**”，即可进行共享Internet连接。
