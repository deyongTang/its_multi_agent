## 标题
Windows XP 的鼠标“单击锁定”功能如何实现?

## 问题描述
Windows XP 的鼠标“单击锁定”功能实现的操作指导。

## 分类
主类别: 接口与外置设备
子类别: 鼠标

Windows XP, 单机锁定, 鼠标, 功能

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
 在[windows](/detail/kd_17391.html) XP操作系统有一项新功能：鼠标单击锁定。用户如果想用鼠标拖拽某个图标或是对话框，可以先点鼠标左键片刻，然后松开左键，移动鼠标就会发现图标或对话框会随之移动，当再次单击鼠标时就停止移动。这样可以更轻松地做鼠标拖拽的操作。具体做法如下：

1、打开“[控制面板](/detail/kd_18077.html)”，选择“鼠标”；（如果是分类视图的控制面板，选择“[打印机](/detail/kd_18076.html)和其他[硬件](/detail/kd_17962.html)”－“鼠标”）

2、选中“启用单击锁定”，在设置按钮可以调整鼠标被按下多久可以“锁定单击”；

3、确定后就可以实现“单击锁定”了。
