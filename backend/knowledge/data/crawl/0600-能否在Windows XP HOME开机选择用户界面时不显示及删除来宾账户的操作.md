# 知识库 600

## 标题
能否在Windows XP HOME开机选择用户界面时不显示及删除来宾账户的操作

## 问题描述
关于在Windows XP系统中来宾账户的相关设置。

## 分类
主类别: 操作系统故障
子类别: 账户相关

## 关键词
软件, 功能, 技巧, Windows, 硬件

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
  Guest帐户为[xp](/detail/kd_17891.html)内置帐户，不能删除。但是xp[启动](/detail/kd_17514.html)后的选择用户界面可以通过以下方法不让guest帐户出现：打开[控制面板](/detail/kd_18077.html)－用户帐户－来宾帐户-禁用来宾帐户。
