# 知识库 742

## 标题
电子词典LN4000操作汇总

## 问题描述
电子词典是一种将传统的印刷词典转成数码方式、进行快速查询的数字学习工具。电脑辞典以轻便易携、查询快捷、功能丰富等特点，成为21世纪学生学习生活、社会人士移动办公的掌上利器。那么电子词典中那么多的功能我们该如何运用呢？本文以LN4000为例为我们演示电子词典的操作方法。
