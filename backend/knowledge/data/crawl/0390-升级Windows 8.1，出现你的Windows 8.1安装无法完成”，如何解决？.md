# 知识库 390

## 标题
升级Windows 8.1，出现你的Windows 8.1安装无法完成”，如何解决？

## 问题描述
下面将指导通过应用商店升级Windows 8.1，出现报错“你的Windows 8.1安装无法完成”的方法。

## 分类
主类别: 操作系统故障
子类别: 系统安装与升级

## 关键词
特色, 功能, 固件, 升级, 系统升级, 最新, 无法完成, Windows 8.1

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
首先确认下载应用商店中的应用和Windows 自动更新功能正常。  
  
1、按键盘上的Windows[徽标键](/detail/kd_17845.html)+X，在弹出菜单上点击“**[命令提示符](/detail/kd_17641.html)（管理员）**”；如图所示：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/b0af8edb833e709e6415b42678954ed8-1714361131588.png&name=mceclip0.png)

2、在命令提示符窗口中输入“**ws[reset](/detail/kd_17978.html)**”，然后按键盘上的回车键（ENTER）；如图所示：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/af667c3a51ef21df3cca3d86aedc13c7-1714361153573.png&name=mceclip1.png)

3、应用商店会被打开，并提示“**已清除应用商店的[缓存](/detail/kd_17389.html)。现在可以在应用商店中浏览应用**”；如图所示：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/a89cbe19b7a1e8f728ae1aec2abd8dfc-1714361165863.png&name=mceclip2.png)

4、重新回到应用商店首页，再次下载Windows 8.1即可。
