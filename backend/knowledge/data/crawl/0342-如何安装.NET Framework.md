# 知识库 342

## 标题
如何安装.NET Framework

## 问题描述
Microsoft .NET Framework是用于Windows的新托管代码编程模型，下面将指导如何安装.NET Framework。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
软件, 功能, 技巧, Windows, 组件, .NET Framework

## 元信息
创建时间:2025-07-10|版本:3.0

## 解决方案
**相关文档：**

[Win10下安装Microsoft .NET Framework 3.5](https://iknow.lenovo.com.cn/detail/135581)

[无法升级.NET Framework 3.5.1](https://iknow.lenovo.com.cn/detail/078867)

[用微软cleanup\_tool清除Microsoft .NET Framework](https://iknow.lenovo.com.cn/detail/320263)

[.NET Framework 3.5安装错误](https://iknow.lenovo.com.cn/detail/125506)

一、Windows 7操作系统

1、[**点击此处**](http://download.microsoft.com/download/9/5/A/95A9616B-7A37-4AF6-BC36-D6EA96C8DAAE/dotNetFx40_Full_x86_x64.exe)下载 “运行”安装.Net FrameWork 4.0；

2、勾选“**我已阅读并接受许可条款**” 然后单击“**安装**”等待安装完成即可。

二、Windows XP操作系统

**注：对于Windows XP系统，目前常用的有以下两个版本,如果您无法确定需要安装哪个版本，可以同时安装两个版本。**  
  
1、**[点击此处](http://download.microsoft.com/download/0/8/c/08c19fa4-4c4f-4ffb-9d6c-150906578c9e/NetFx20SP1_x86.exe)**选 “**运行**”开始安装下载安装.Net FrameWork 2.0

**[点击此处](http://download.microsoft.com/download/0/6/1/061F001C-8752-4600-A198-53214C69B51F/dotnetfx35setup.exe)**选 “**运行**”开始安装下载安装.Net FrameWork 3.5  
  
  
2、选择“**我已阅读并接受许可条款（A）**” 然后单击“**安装**”等待安装完成即可。
