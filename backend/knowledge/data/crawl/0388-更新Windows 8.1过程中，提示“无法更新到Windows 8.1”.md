# 知识库 388

## 标题
更新Windows 8.1过程中，提示“无法更新到Windows 8.1”

## 问题描述
更新Windows 8.1过程中，提示“无法更新到Windows 8.1”，首先请确认您的机型是否支持升级到Windows 8.1。

## 分类
主类别: 操作系统故障
子类别: 系统安装与升级

## 关键词
升级win8.1, 无法更新, 提示, 软件, 功能, 技巧

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
首先请确认您的机型是否支持升级到[Windows 8](/detail/kd_17985.html).1。  
  
**一、支持[Win8](/detail/kd_17986.html).1的[硬件](/detail/kd_17962.html)要求**  
  
[Windows 8.1](/detail/kd_17987.html)能够在支持 Windows 8 的相同硬件上出色运行：  
  
[处理器](/detail/kd_17477.html)： 1 GHz 或更快  
  
[内存](/detail/kd_17351.html)： 1 GB（32 位）或 2 GB（64 位）  
  
可用[硬盘](/detail/kd_17342.html)空间： 16 GB（32 位）或 20 GB（64 位）  
  
[图形卡](/detail/kd_17476.html)：带有 WDDM [驱动](/detail/kd_17488.html)程序的 MicrosoftDirectX 9 图形设备  
  
若要使用某些特定功能，还需要满足以下附加要求  
  
对于 Windows 8.1 的 64 位安装，你的 [CPU](/detail/kd_17345.html) 还必须支持 CMPXCHG16b、PrefetchW 和 LAHF/SAHF。  
  
若要使用触控，你需要支持[多点触控](/detail/kd_17478.html)的平板或显示器。  
  
若要访问 Windows 应用商店并下载和运行应用，你需要有效的 Internet 连接、至少 1024 x 768 的屏幕[分辨率](/detail/kd_17627.html)以及 Microsoft 帐户。

Internet 接入（可能产生 [ISP](/detail/kd_17479.html) 费用）。
