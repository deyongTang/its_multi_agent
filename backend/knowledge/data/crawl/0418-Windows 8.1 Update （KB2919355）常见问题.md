# 知识库 418

## 标题
Windows 8.1 Update （KB2919355）常见问题

## 问题描述
从Windows 8升级到Windows 8.1，以及Windows 8.1的Update更新中会存在很多我们所不了解的问题，那么该如何解决呢？一起来看下本文的汇总……

子类别: 系统应用操作

## 关键词

## 解决方案
**1、需要先安装的更新KB2919442 是否可以在Windows Store 下载?**

答：不会。自三月11日起，它可以从[**Windows 更新**](https://support.microsoft.com/zh-cn/kb/2919442)获得。

**2、微软是否有计划在网站上提供Windows 8.1 Update 的信息呢?**

答：是的。Windows 在线网站将提供有关内容，包括完成Windows 8.1 Update更新需要的前提条件，及Windows 8.1 Update中的新功能。

**3、我的电脑是Windows 8, 还没升级到Windows 8.1, 我可以直接升级到Windows 8.1 update 吗？还是我需要先升级到Windows 8.1 然后再升级到8.1 update?**

答：自2014年4月8日起，Windows 8 用户可以通过Windows Store直接升级到Windows 8.1 + Update。

**4、升级文件的大小是600 MB吗？为什么这比要求的可用空间还要大？**

答：进行升级时通过Windows更新下载的文件大小最大为657 MB。如果电脑已经安装了之前发放的更新，那么下载的文件大小将显著减少，只有107 MB。

**5、如果Windows8.1的用户没有升级到Windows8.1 update，是否未来仅是安全更新不可以再安装了呢？**

答：Security Definition updates仍可进行。非安全性更新取决于各自的要求。但是所有Windows 更新都要求Windows 8.1 Update，Windows Defender 仍会收到definition 更新。

**6、Windows 8.1 Update 是被看作一个传统的service pack 吗?**

答：不是，它不是一个service pack，它是一个为Windows推送更行的新形式，也是Windows更新的新形式。但是它也只是每月第二个周二补丁日的一个大的更新。传统的service pack不包含产品功能的改变。这次的Windows 8.1 update确实有产品功能变化。

**7、Windows 8.1 update 如何通知给用户?**

答：Windows 8.1 Update 信息将会通过Windows Blog、Windows在线和TechNet网站对用户沟通。一些Windows 8和Windows 8.1的用户还会收到邮件通知。

**8、Windows 8用户是否需要升级到Windows 8.1 Update 以获得未来安全更新呢?**

答：不用。

**10、是否会有媒体宣传？**

答：是的。在Windows Blog上会有发布信息，并通过Windows在线网站和TechNet向用户沟通。也将有其他的宣传方式。

**11、如何不喜欢更新其Windows 8.1的用户解释他们必须进行更新呢？因为微软不再提供安全更新吗？**

答：Windows 8.1 Update是Windows 8.1的必需安全更新，它为用户提供安全性和增强功能以保持个人电脑的安全，特别是对鼠标和键盘的用户提高了整体可用性的体验。

**12、如果机器自带Windows8.1，然后安装Windows 8.1 update, 在机器重置后会移除Windows 8.1 update吗？**

答：是的，如果进行了重置，电脑将恢复到初始重置镜像所能完成的重置/恢复，这时的重置/恢复分区不会进行Windows 8.1 update更新。

**13、Windows 8.1 update更新后，产品密钥或激活有变化吗？**

答：没有。

**14、会对客户进行关于Windows 8.1 update可用性的通知吗？**

答：更新将会分批进行，所以无法在同一时间提供给大家，客户可以手动搜索Windows Update看它是否可用。

**15、从Windows 8.1 升级到Windows 8.1 update是否存在已知的驱动问题？**

答：没有。任何与Windows 8.1兼容的驱动都与Windows 8.1 Update兼容。

**16、与Windows 7和Windows8.1安装在一个硬盘驱动器的双启动兼容性有任何的更改吗？**

答：没有。对启动设置数据没有更改。

**17、是不是所有版本的Windows 8都可以升级到Windows 8.1？**

答：是的。企业版本和批量许可版本的更新由系统管理员处理这些版本的安装。

**18、可以在复选框中取消非平板配置，让系统启动到开始屏幕而不是桌面吗？**

答：是的。在任务栏属性里的导航选项卡中，取消选中“当我登陆或关闭屏幕上的所有应用时，转到桌面而不是”开始”屏幕”选项。

**19、是否可以下载Windows 8.1 update后，离线后执行更新？**

答：可以。微软下载中心提供此更新，需下载和安装5个不同的更新完成此操作。

**20、是否可能在更新后进行平行安装（parallel installation），以防客户遇到无法启动的情况？**

答：可以。建议用户尝试做一个系统刷新，保留数据，重置底层 OS。

**21、Windows 8.1 update 会从商店可用还是从Windows Update推出？**

答：此更新会像每月更新一样从Windows Update推出给Windows 8.1的用户使用。Windows 8.0的用户会从2014年4月8日起开始在商店中收到Windows 8.1 + update更新。

**22、在安装更新后，还能使用系统还原回到安装前一天吗？**

答：您可以卸载，但是因为多种原因，我们不推荐用户卸载更新。卸载可能会丢失很多功能，影响以后的更新服务并会影响OneDrive的功能。

**23、如果用户不小心卸载了更新，可以重新它吗？**

答：可以。

**24、在安装KB2919355（Windows 8.1 Update）后，硬盘分区引导记录所使用的操作系统映像会被更新吗？**

答：重置会让用户回到他们初始的恢复镜像，如果机器自带Windows 8.1，他会回到Windows 8.1 RTM。更新不会改变恢复镜像。

**25、“高级启动选项”有什么变化吗？**

答：没有变化。

**26、更新后通过F8功能键进入WinRE是否被影响？**

答：在升级到Win8.1 update后进入WinRE中是和Windows 8.1时一样的。

**27、如果遇到激活问题，更新到Windows 8.1 update后，Windows 8的命令仍然可用吗？**

答：是的。

**28、此更新在发布给客户前，是否可供实验室测试使用呢？**

答：是的。最终发布Windows 8.1 update版本已交付给OEM工程团队进行测试。您可以通过联系您的工程团队来获取这个更新。

**29、微软会对Windows 8.1 update更新有问题或需要安装帮助的客户提供免费支持吗？**

答：是的。微软会对安装Windows 8.1 update需要帮助的客户提供免费支持。

**30、微软对Windows 8.1 update的支持范围是什么？**

答：微软会对使用Windows 8.1 Update更新电脑时需要帮助的用户提供免费支持。微软消费者支持分为技术支持和安装下载与设置支持。我们会对技术支持进行质保检查，但提供免费的安装下载与设置支持。
