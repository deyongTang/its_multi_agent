# 知识库 809

## 标题
联想ET960电脑手机每次复位或者冷启动后，关闭视频声音或者不播放视频？

## 问题描述
ET960电脑手机每次复位或者冷启动后，关闭视频声音或者不播放视频设置。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
复位, 冷启动, 视频声音, ET960

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
联想ET960电脑手机每次复位或者[冷](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17399)[启动](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17514)后，总是会播放一段[视频](http://iknow.lenovo.com/knowledgeDetail.html?knowledgeId=17880)，如果想关闭该视频的声音，只需在播放时向下拨动一下滚轮键即可！  
  
如果不想播放视频，可以先设置主人信息的“标识”、“附注”，再在主人信息中的“选项”中，将“标识”或者“附注”选中即可！这样每次复位或者冷启动后就只会显示主人信息，而不会播放视频了！
