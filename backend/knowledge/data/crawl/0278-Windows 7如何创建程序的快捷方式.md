# 知识库 278

## 标题
Windows 7如何创建程序的快捷方式

## 问题描述
Windows 7创建程序的快捷方式的操作指导。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
Windows 7, 设置, 管理, Windows, 防火墙, 安全, 杀毒软件, 网络, 图表, 快捷方式

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、找到安装应用程序的安装路径，然后直接将应用程序的图标右击选择发送桌面快捷方式。

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/e618e00f7904e3065108d398a3b88470-1714297400956.png&name=mceclip0.png)

2、桌面空白右击新建“**快捷方式**” 然后弹出一个框，你可以在点击“**浏览**”，然后选择你所知道的安装后路径，找到该文件或文件夹后点击“**确定**”然后点击下一步，输入“**快捷方式名称**”点击“**完成**”。

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/a8b6770b8e08baa8f3d8c3e24da59cf5-1714297431512.png&name=mceclip1.png)

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/553eb8e2816dc32ba224cf77db6eb9cf-1714297447150.png&name=mceclip2.png)

注：如果是创建某文件的快捷方式，方法和创建程序的快捷方式相同，区别是选择到某文件即可。
