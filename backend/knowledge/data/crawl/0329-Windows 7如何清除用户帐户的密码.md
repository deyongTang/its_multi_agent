# 知识库 329

## 标题
Windows 7如何清除用户帐户的密码

## 问题描述
Windows 7每次开机都需要输入用户帐户密码的解决方案。

## 分类
主类别: 操作系统故障
子类别: 账户相关

## 关键词
密码, 管理, 设置, 显示, 硬件, Windows 7, 查看, 清除

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、打开开始菜单按钮，然后选择**“控制面板”**；

![教大家win7开机密码怎么取消](https://img.php.cn/upload/article/000/465/014/168879919430973.png)

2、点击“**添加或删除用户帐户**”；

![教大家win7开机密码怎么取消](https://img.php.cn/upload/article/000/465/014/168879919552570.png)

3、点击需要清除密码的帐户。注意当前使用的帐户，可以清除自己帐户的密码。如果需要清除其他帐户的密码，当前使用的帐户需要有管理员权限。

![教大家win7开机密码怎么取消](https://img.php.cn/upload/article/000/465/014/168879919595592.png)

4、点击“**删除密码**”弹出对话框，再点击“**删除密码**”即可删除此帐户的密码。

![教大家win7开机密码怎么取消](https://img.php.cn/upload/article/000/465/014/168879919535898.png)

![教大家win7开机密码怎么取消](https://img.php.cn/upload/article/000/465/014/168879919663825.png)

注意：如果电脑本身还有其他安全机制，如：指纹识别、Port Locker、联想安全中心等关于安全的[软件](/detail/kd_17963.html)，可能需要先卸载这些安全软件后，才可以清除Windows帐户密码。
