# 知识库 319

## 标题
如何进入Windows高级选项菜单

## 问题描述
Windows 8/8.1 取消了传统的F8进入高级选项菜单模式，使得进入安全模式和选择其他高级选项成为了一件难事，那么如何正确操作进入到新系统的高级选项菜单呢？

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
高级菜单, 高级选项, windows, 安全模式

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
**Windows XP 和 Windows 7系统如下所示：**  
  
1、打开或重新启动电脑，当出现"**Lenovo**”界面时，连续点击"**F8**”键，如下图：

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/201.files/image001.jpg)

2、进入到&ldquo;**Windows高级选项**”菜单。用上下方向键移动高亮显示条到所要操作系统的工作模式（如：安全模式、启用VGA模式、最后一次正确的配置等），然后按"**Enter**”键进入，根据屏幕提示操作即可。  
  
Windows XP操作系统下

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/200.files/image002.jpg)

Windows Vista操作系统下

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/200.files/image002.jpg)

Windows 7 操作系统下

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/201.files/image002.jpg)

**Windows 8系统如下所示**  
  
**根据电脑开机的实际情况参考以下步骤：**  
  
**若开机已经无法进入Windows 8桌面**  
  
请按住"电源按钮”保持5秒左右支持系统关机；再按"电源按钮”将电脑开机，重复3次左右，进入恢复界面，点击"高级恢复选项”后，然后（跳转下面第4步）；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559316049412.jpg)  
  
  
  
**若开机能进入Windows 8桌面**  
  
1、在开机界面点击metro界面"桌面图标”进入桌面；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559318005175.jpg)

2、鼠标放在右下角，出现超级按钮，点击"**设置**”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559319055421.jpg)

3、点击电源，此时按住shift，然后点击"**重启**”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559320010270.jpg)

4、请稍后，出来一个界面，选择"**疑难解答**”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559320005132.jpg)

5、在"**疑难解答**”下面选择"**高级选项**”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559321080426.jpg)

6、"**高级启动**”下会有五个选项，选择"**Windows启动设置**”。

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559322092484.jpg)

7、出现Windows启动设置，选择"**重启**”，如下图：

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/1498209559322015966.jpg)

8、如果需要进入"**安全模式**”，重启选择"**4、安全模式**”，即可。如下图：

![20150808155751008.png](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20180425/1524636397013072610.png)

**Windows 10系统如下所示**

**分两种情形:**系统可以启动到Windows桌面，以及系统无法启动到Windows桌面

**Windows 10可以启动到桌面**[**点击这里**](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/203.html#yxlm)

**Windows 10已经无法启动到桌面**[**点击这里**](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/203.html#wfqd)

**情形一：Windows 10可以启动到桌面**

1、进入Windows 10桌面后，点击开始菜单，然后再点击“设置”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155749001.png)

2、然后选择“更新和安全”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155749002.png)

3、在“更新和恢复”界面下点击“恢复”，然后在高级启动下面点击“立即重启”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750003.png)

4、选择一个选项，选择“疑难解答”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750004.png)

5、在“疑难解答”中点“高级选项”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750005.png)

6、在高级选项中点击“启动设置”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750006.jpg)

7、点击“重启”按钮；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750007.jpg)

8、电脑此时会重启，重启后即可进入Windows高级选项菜单界面，如下图所示：

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155751008.png)

**情形二：Windows 10已经无法启动到桌面**

1、开机过程到无法继续启动画面时，长按电源键关机，并重新开机，如此重复3次以上，这样再次开机后直接会出现“恢复”的界面，单击“查看高级修复选项”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155751010.jpg)

2、选择一个选项，选择“疑难解答”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750004.png)

3、在“疑难解答”中点“高级选项”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155750005.png)

4、在高级选项中点击“启动设置”；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155751011.jpg)

5、点击“重启”按钮；

![](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20170623/20150808155751012.jpg)

6、电脑此时会重启，重启后即可进入Windows高级启动菜单界面，如下图所示：

![20150808155751008.png](https://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/images/20180425/1524636329071060201.png)
