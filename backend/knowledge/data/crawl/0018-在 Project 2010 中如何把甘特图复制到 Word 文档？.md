# 知识库 18

## 标题
在 Project 2010 中如何把甘特图复制到 Word 文档？

## 问题描述
有些同学在一开始使用 Project 2010 时，要把甘特图复制到 Word 中，却一下子找不到复制图片的功能，只能用 Windows 7 自带的截图工具。下面再指导在 Project 2010 中如何把甘特图复制到 Word 文档的方法。

## 分类
主类别: 预装软件
子类别: Office软件

## 关键词
Project 2010, 甘特图, Word 文档, 复制

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
**故障现象：**有些同学在一开始使用 Project 2010 时，要把甘特图复制到 Word 中，却一下子找不到复制图片的功能，只能用 Windows 7 自带的截图工具。

**解决方案：**

1.打开 Project 2010，在“**任务**”选项卡里的“**剪贴板**”组里，点击“**复制**”按钮旁边的向下小箭头，你就能看见“**复制**”和“**复制图片**”两个按钮。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120604/1338795940359_701.jpg)

2.单击“**复制图片**”，在“**复制图片**”对话框中设置完成，点击“**确定**”即可复制粘贴到 Word 文档中。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120604/1338795940359_66.JPG)

可能有些同学还是觉得比较麻烦，有个更方便的方法不妨试试：

1.打开 Project 2010，单击“**文件**”按钮，然后点击“**选项**”。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120604/1338795940359_254.jpg)

2.在“**Project 选项**”对话框中，单击“**自定义功能区**”，然后点击“**新建组**”按钮。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120604/1338795940359_975.JPG)

3.在“**常用命令**”中，选中“**复制图片**”，单击“**添加**”。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120604/1338795940359_529.JPG)

4.然后单击上下小箭头，可以调整“**新建组（自定义）**”组所在的位置，你还可以根据你的喜好来重命名新建组的名称。最后点击“**确定**”，单独的“**复制图片**”按钮就在“**任务**”选项卡中出现，方便快捷。

![](https://webdoc.lenovo.com.cn/lenovowsi/20120604/1338795940359_61.JPG)
