# 知识库 406

## 标题
开机时，需要按F1（或F2）键后才能继续启动

## 问题描述
开机时，需要按F1（或F2）键后才能继续启动的几种现象分析。

## 分类
主类别: BIOS
子类别: BIOS设置/模拟

## 关键词
开机, 启动, F1, F2

## 元信息
创建时间:2025-09-11|版本:3.0

## 解决方案
开机时，需要按F1（或F2）键后才能继续启动，请核实属于以下哪种报错现象：  
  
**现象1**、开机自检时，报“**Pri Master Hard Disk:S.M.A.R.T. Status BAD,Backup and Replace Press F2 to Resume**”  
  
此故障为硬盘故障，建议及时备份数据并报修。

**现象2**、开机自检时，报“**SMART Failure Predicted on Hard Disk，Press F1 to Continue**”  
  
此故障为硬盘故障，建议及时备份数据并报修。

**现象3**、开机自检时，报“**CMOS Date/Time Not Set，Press F2 to Run SETUP，Press F1 to load default values andcontinue**”  
  
如果笔记本有此报错，请开机并进入BIOS设置，按F9恢复默认值后，重新设置CMOS日期及时间，按F10保存退出后，关机断电并拔掉电池，放置一段时间后再开机测试；如无效请在此界面按F1，进入系统后升级杀毒软件，全面杀毒；如依然无效，请您报修。

**现象4**、开机自检时，报“**0271：Check date and time settings ，Press F1 to resume， F2 to Setup**”  
  
如果笔记本有此报错，请开机并进入BIOS设置，按F9恢复默认值后，重新设置CMOS日期及时间，按F10保存退出后，关机断电并拔掉电池，放置一段时间后再开机测试；如无效请在此界面按F1，进入系统后升级杀毒软件，全面杀毒；如依然无效，请您报修。

**现象5**、开机自检时，报“**02F0:CPU ID:55 No Microcode Update Loaded for Alternate Processor**”  
  
请检查并拔出所有的外接设备，如存储卡，读卡器等USB设备，同时，如果您有自行升级过硬件，请复原后测试；如无效，请开机并进入BIOS设置，按F9恢复默认值后，按F10保存退出，关机再开机测试；如依然无效，请您报修。
