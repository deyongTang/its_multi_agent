# 知识库 372

## 标题
Windows 7 Windows 8下如何打开麦克风

## 问题描述
Windows 7和Windows 8下如何打开麦克风，进而实现录音或者语音通话的操作指导。

## 分类
主类别: 内置设备
子类别: 麦克风

## 关键词
麦克风, 录音, win7, win8, 录音, 语音

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
1、右键单击任务栏上的“小喇叭”图标，选择“录音设备”，如图：

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/34.files/image001.jpg)

2、右键单击空白处，单击“显示禁用的设备”，如图：

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/34.files/image002.jpg)

3、右键单击已出现的已停用的“[麦克风](/detail/kd_17501.html)”，单击“启用”，如图：

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/34.files/image003.jpg)

4、可以看到“麦克风”已经成为默认设备，问题应该可以解决。如图：

![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/34.files/image004.jpg)
