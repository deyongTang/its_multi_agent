# 知识库 322

## 标题
Windows 7 操作系统禁用“自动重新启动”功能

## 问题描述
在机器出现故障时，系统禁用“自动重新启动”功能可观察故障信息，方便分析故障原因，下面将指导Windows 7 操作系统禁用“自动重新启动”功能。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
自动重新启动, 管理, 设置, 显示, 硬件, Windows 7, 查看

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、鼠标左键单击桌面左下角的“**开始**”按钮，在弹出菜单中的“**计算机**”点击鼠标右键，然后选择“**属性（R）**”；

2、在弹出的系统窗口，点击左侧列表的“**高级系统设置**”；

3、在弹出的系统属性对话框，点击[启动](/detail/kd_17514.html)和故障恢复下的“**设置（T）…**”按钮；

4、在弹出的启动和故障恢复对话框，取消勾选“**自动重新启动（R）**”选项，最后点击“**确定**”按钮。

5、设置完毕后，重新启动即可生效。
