# 知识库 848

## 标题
如何排除由主板ACPI不兼容报错所造成无法安装WIN2000SERVER的问题？

## 问题描述
安装windows 2000 srever 排除由于主板ACPI不兼容报错方法。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
ACPI, WIN2000, 报错, 驱动

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
**故障描述：**

       当安装WIN2000/WIN2000SERVER时，发生蓝底白字"The ACPI Bios in this [sys](/detail/kd_17769.html)tem is not fully compliant with the ACPI specification..."

**解决方案：**

1、重新开机按下F7就可离开ACPI MODE SETUP到TEXT MODE SETUP就可以继续安装WIN2000/WIN2000SERVER。

2、如果担心会有规格不符的状况也可以在WIN2000/WIN2000SERVER安装时，按F5会出现四个选项；

3、若没有厂商提供的[驱动](/detail/kd_17488.html)程序，请选择standard pc，完成WIN2000/WIN2000SERVER安装。
