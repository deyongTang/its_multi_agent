# 知识库 771

## 标题
如何设置Windows XP在启动时自动登录本机

## 问题描述
我们在Windows XP系统下需要在启动时自动登录本机，将如何设置？

## 分类
主类别: 操作系统故障
子类别: 系统登录问题

## 关键词
XP, 自动登录, 启动, 本机

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
让Windows XP实现自动登录，只需按如下操作：   

单击“开始--运行”，在运行窗口中输入Control userpasswords2，可以打开用户帐户窗口，如下图：

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-12-30/EO5MdE2kpz7J0keb.bmp)  
  
在窗口中将“要使用本机，用户必须输入用户名和密码”选项取消选择，随即弹出如下窗口，确定用户名和密码后即可完成操作。  
  
![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2004-12-30/j7vk7Rv9RK0P7ckg.bmp)
