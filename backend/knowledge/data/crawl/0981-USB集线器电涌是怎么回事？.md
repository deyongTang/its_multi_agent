# 知识库 981

## 标题
USB集线器电涌是怎么回事？

## 问题描述
电脑右下角会出现集线器端口上的电涌警告的解决方法。

## 分类
主类别: 接口与外置设备
子类别: USB接口相关

## 解决方案
**问题描述**
--------

把USB外接硬盘盒接进电脑，有时电脑右下角会出现集线器端口上的电涌警告。有时伴随移动硬盘发出咔咔声，指示灯忽明忽灭，电脑无法找到该设备的现象。

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-05-12/cleivCW88qf4v49f.JPG)

**解决方案**
--------

1、针对集线器端口上的电涌警告：在“设备管理器” --> “通用串行总线控制器”－“USB Root Hub” --> “电源”中可以看到，正在使用的USB设备的功耗。由于频繁的插拔USB设备，或在USB设备传输数据时将其拔出，此时容易造成USB接口的瞬时电流过大，从而报错。平时注意安全地卸载USB设备就不会有问题。Windows系统中，可以通过点击任务栏的通知区域，选择“安全删除硬件”来完成。这样可以确保在拔出设备时不会产生瞬时大电流。  
  
2、确认USB驱动程序安装正确无误。如果驱动程序有问题，可能会导致设备与系统的通信不稳定，从而引发电涌问题。  
   
3、USB-IF规定USB端口提供的电源是5V/500mA，但目前不少外设所需电源超过此标准。尤其是外置硬盘盒或移动硬盘等设备，使用时很容易出现此情况。有时使用了USB延长线，或同时使用多个USB设备，也可能造成USB供电不足。如果USB设备所需电源超过USB端口的提供能力，我们可以考虑为USB设备接上专用的变压器；采用从电脑PS/2口与USB口的取电方式增强供电；更换USB硬盘盒或换用低转速的硬盘 ；使用双USB插头供电的USB硬盘盒；不使用USB延长线，USB设备应插在机器后面的USB接口以避免信号衰减 ；不要同时使用多个USB设备，一般可以解决该问题。另外还可以通过加装一个有源USB HUB解决这个问题。

4、针对移动硬盘发出咔咔声：USB设备的供电需求超过了500mA，此时点击USB图标后再选择“复位”，USB设备有可能被正确识别并使用。

预防措施
----

定期清理和维护USB接口：定期清理USB接口上的灰尘和污垢，保持接口的清洁和干燥。

避免在恶劣环境下使用USB设备：避免在潮湿、高温或电磁干扰严重的环境下使用USB设备。

遵循正确的插拔顺序：在连接或断开USB设备时，应遵循正确的插拔顺序。首先连接设备的USB接口，然后再接通电源；在断开设备时，应先断开电源，再拔出USB接口。
