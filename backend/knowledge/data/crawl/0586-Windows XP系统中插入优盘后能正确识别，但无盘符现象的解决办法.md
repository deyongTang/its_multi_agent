## 标题
Windows XP系统中插入优盘后能正确识别，但无盘符现象的解决办法

## 问题描述
winxp系统中插入优盘后能正确识别，但无盘符现象的解决办法。

## 解决方案
XP系统在安装[sp2](/detail/kd_17484.html)以后插入U盘，能正确识别，但是资源管理器里却没有盘符，而其他[USB](/detail/kd_17416.html)设备（如：鼠标）则没有故障，更换U盘测试，故障相同,主要的原因是设置的一个网络[硬盘](/detail/kd_17342.html)的盘符与分配给U盘的盘符相同,解决的方法很简单,可以参照以下步骤

1.首先鼠标右键点击 "我的电脑",然后选择"管理"进入计算机管理（如图）

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-06-22/hLGkenyYsB70u2rm.JPG)

2.进入"计算机管理后",单击左面的"[磁盘](/detail/kd_17452.html)管理"选项,在右侧会显示计算机上所有的盘符,找到可移动磁盘（如图）

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-06-22/BNnK127uvfh32G1t.JPG)

3.最后鼠标右键点击"可移动磁盘",选择更改[驱动](/detail/kd_17488.html)器名和路径,将盘符改名即可（图中显示I盘可以改成J盘）
