## 标题
Windows Phone 日历相关设置简介

## 问题描述
Windows Phone 的日历包含手机自带日历，以及添加的邮箱帐号日历。本文将介绍如何对日历进行设置，如农历开关、显示时间范围切换（天/月），不同邮箱对应的日历区分颜色、标识出好友的生日、标识出节假日等。

## 分类
主类别: 预装软件
子类别: 手机应用

## 关键词
Windows Phone, 日历, 设置

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
 Windows Phone 的日历包含手机自带日历，以及添加的邮箱帐号日历。本文将介绍如何对日历进行设置，如农历开关、显示时间范围切换（天/月），不同邮箱对应的日历区分颜色、标识出好友的生日、标识出节假日等。

1、打开日历

在“**开始屏幕**”或者已安装应用列表中点击“**日历**”，打开日历界面：

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_15.png)

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_358.png)

 

2、显示农历

点击“**…**”按钮，在菜单中点击选项“**显示农历**”即可显示农历信息：

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_577.png)

 

显示农历后，可按照同样的方式在菜单中点击“**隐藏农历**”而隐藏农历信息。

3、显示时间范围切换

在屏幕下方的菜单栏中，选择“**今天**”或者日历图标，可以在“**今天**”和“**月份**”之间切换显示的时间范围。如下图所示：

 

 ![](https://support.microsoft.com/Library/Images/2732446.png)

 

点击手机硬件左向箭头按钮即可返回“**今天**”视图。

4、不同邮箱的日历显示不同的颜色

在菜单中选择“**设置**”：

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_272.png)

 

在日历的“**设置**”界面，可以为每一个日历设置不同的颜色：

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_410.png)

 

点击颜色框，在弹出的“**选择颜色**”对话框中选择某一颜色即可为对应的日历设置颜色。不同日历中的待办事项和提醒等在日历中显示不同的文字颜色，如下图所示：

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_520.png)

 

5、标识出好友生日和节假日

Windows Phone 设置 Windows Live 帐号后，Live 邮箱会自动下载节假日日历以及好友生日日历等到手机日历功能中；我们只要在日历设置中，将这些日历功能打开后，在日历中则会显示相应的信息。

 

![](https://webdoc.lenovo.com.cn/lenovowsi/20120711/1344225928060_875.png)

 

如上图所示，将“**生日日历**”与“**中国假日**”日历开启后在日历中即会标识出相应的信息，如“**10月1日**”会显示“**国庆节**”。
