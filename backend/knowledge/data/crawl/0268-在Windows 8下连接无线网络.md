# 知识库 268

## 标题
在Windows 8下连接无线网络

## 问题描述
本文提供了在Windows 8下连接无线网络的步骤，希望对您有所帮助。

## 分类
主类别: 网络问题
子类别: 无线连不上

## 关键词
设置, 管理, Windows, 网络, Windows 8, 无线网络, 网络连接

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
1、点击右下角网络连接图标；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/1.3_ljwxwl_Windows_8_SWS_WEB_WX.files/image001.jpg)

2、此时屏幕右侧会列出当前搜索到的网络连接。选择要连接的网络，然后单击；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/1.3_ljwxwl_Windows_8_SWS_WEB_WX.files/image002.jpg)

3、选择“**连接**”；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/1.3_ljwxwl_Windows_8_SWS_WEB_WX.files/image003.jpg)

4、输入密码后点击**下一步**；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/1.3_ljwxwl_Windows_8_SWS_WEB_WX.files/image004.jpg)

5、根据实际的网络情况选择网络环境；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/1.3_ljwxwl_Windows_8_SWS_WEB_WX.files/image005.jpg)

6、连接成功！

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/1.3_ljwxwl_Windows_8_SWS_WEB_WX.files/image006.jpg)
