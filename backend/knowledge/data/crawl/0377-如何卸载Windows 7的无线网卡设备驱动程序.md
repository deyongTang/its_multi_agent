# 知识库 377

## 标题
如何卸载Windows 7的无线网卡设备驱动程序

## 问题描述
在Windows 7系统下，有时我们的无线网络出现故障需要卸载无线网卡的驱动程序进行调试或安装，该如何操作呢？

## 分类
主类别: 其他硬件相关知识
子类别: 驱动相关

## 关键词
Windows 7, 无线网卡, 驱动, Wireless, Lan, Wifi

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
**方法一：**

1、 双击打开“计算机”图标；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/348_file/54691392959532.png "clip_image001.png")

2、单击“卸载或更改程序”；

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/348_file/97081392959532.png "clip_image002.png")

3、选择需要卸载的驱动程序（此处以卸载Intel 1000N无线网卡驱动为例），点击鼠标右键，选择“卸载”后，按照屏幕提示完成卸载。

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/348_file/78061392959532.jpg "clip_image003.jpg")

**备注**：Atheros无线网卡一般是名字带有 Atheros wlan Client 的，Broadcom无线网卡一般是名字带有Broadcom Wireless [LAN](/detail/kd_17375.html) Driver，其他型号网卡名字通

常是lenovo\_wireless\_driver的软件。

**方法二：**

1、在计算机图标上点击鼠标右键，选择“管理”；

2、在打开的计算机管理界面（此处以卸载intel 1000N 无线网卡驱动为例），依次点击“设备管理器”，“网络控制器”，在“intel（R） WiFi Link 1000 BGN”设备上点击鼠标右键，选择”卸载”，

3、勾选”删除此设备的驱动程序软件。”后点击确定，按照屏幕提示完成卸载。

![](http://robotterm.ecare365.com/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/348_file/68501392959532.png "clip_image006.png")
