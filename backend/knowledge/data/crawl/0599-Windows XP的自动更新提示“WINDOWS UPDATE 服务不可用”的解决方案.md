# 知识库 599

## 标题
Windows XP的自动更新提示“WINDOWS UPDATE 服务不可用”的解决方案

## 问题描述
Windows XP的自动更新功能无法使用，提示“WINDOWS UPDATE 服务不可用”的解决方案。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
软件, 功能, 技巧, Windows XP, 硬件

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
出现此现象的原因是服务中的AUTOM[ATI](/detail/kd_17407.html)C UPDATE被禁用。解决方法是在[控制面板](/detail/kd_18077.html)的管理工具中打开双击“服务”图标，在服务列表中找到AUTOMATIC UPDATE，点击启用此服务，

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-06-22/3r3FAC1u2HDy2Tq0.JPG)  
  
然后回到系统属性中将自动更新功能开启即可。

![](https://webdoc.lenovo.com.cn/lenovowsi/uploadimages/2005-06-22/ieyPo1A0m6t6AciV.JPG)
