# 知识库 865

## 标题
万全3200C如何清除系统事件日志？

## 问题描述
万全3200C清除系统事件日志。

## 分类
主类别: 操作系统故障
子类别: 开关机问题

## 关键词
日志, 3200C, 开机提示, system event

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
**故障现象：**

开机提示:[sys](/detail/kd_17769.html)tem event logging is full.按F1可跳过，操作系统可正常[启动](/detail/kd_17514.html)和使用。

 

**故障分析：**

系统事件[日志](/detail/kd_17786.html)已满，需要清空，并不影响正常使用。

 
**解决方案:**

开机时按F2进入[BI](/detail/kd_17343.html)[OS](/detail/kd_17441.html)àSYSTEM主菜单àError Log Initalization子菜单à按回车键后系统会显示“Event Log Cleared！“à保存退出即可。
