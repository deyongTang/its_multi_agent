# 知识库 881

## 标题
金山词霸在AMD Athlon 64 CPU机型中运行时导致桌面图标消失的解决方案

## 问题描述
本文提供了金山词霸在AMD Athlon 64 CPU机型中运行时导致桌面图标消失的解决方案。

## 分类
主类别: 操作系统故障
子类别: 非Windows系统支持

## 关键词
金山词霸, AMD, 桌面图标, 消失

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
金山词霸各版本（200220032005）在64位[CPU](/detail/kd_17345.html)（AMD athlon64）的平台上运行，最小化后会出现[windows](/detail/kd_17391.html)系统桌面图标消失不断闪烁的情况词霸2002会出现假[死机](/detail/kd_17517.html)。

测试表明导致此问题的原因是由于金山词霸屏幕取词有[BUG](/detail/kd_17831.html)导致，词霸2005升级到SP1补丁可以解决此问题，词霸2003升级SP3词霸2002升级SP1都无法解决此问题，以上各版本在不升级补丁的情况下关闭屏幕取词后也可以解决此问题。
