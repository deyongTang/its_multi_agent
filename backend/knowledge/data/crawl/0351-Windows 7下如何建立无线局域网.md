# 知识库 351

## 标题
Windows 7下如何建立无线局域网

## 问题描述
本文提供了Windows 7下如何建立无线局域网的方法指导。

## 分类
主类别: 网络问题
子类别: 共享上网

## 关键词
Windows, 无线局域网, 在线, 局域网, 无线网卡, 共享

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
**操作步骤：**

1、启用无线[网卡](/detail/kd_17366.html)：请首先确认[无线网卡](/detail/kd_18121.html)的[驱动](/detail/kd_17488.html)程序已经正确安装，并打开无线网卡天线开关。联想笔记本大部分是FN＋F5（或F4）开启；

2、打开“[控制面板](/detail/kd_18077.html)网络和 Internet网络和[共享](/detail/kd_17685.html)中心”，选择“设置新的连接或网络”；

3、在打开的选项中选择“设置无线临时网络”，点击下一步，如图所示：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/5a0df0b74151f1aca52633d67a66be86-1714378701687.png&name=mceclip0.png)

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/4ed31886d0455566ece465d6fe690fdf-1714378721820.png&name=mceclip1.png)

4、为创建的网络命名并选择合适的安全类型，如图所示：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/a35e906ec79745973fbb26f63ac2306a-1714378748311.png&name=mceclip2.png)

5、无线[局域网](/detail/kd_17376.html)已经可以使用；  
  
（1）如果要在电脑间共享文件，请按照如图提示启用文件共享即可。

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/d1422aa3ad0b6be0a9bc49c3545b6e50-1714378849732.png&name=mceclip3.png)

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/4e2c07ac0aa187cb4e614d2353dd41b8-1714378864097.png&name=mceclip4.png)

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/c8af8c157f703b761411cf11e89501c8-1714378884087.png&name=mceclip5.png)

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/b107ba67bd737aedb2803397b32e7856-1714378901756.png&name=mceclip6.png)

（2）如果要共享Internet连接，请选择“启用Internet连接共享”即可，

6、其他电脑在无线网络列表中选择对应的[SSID](/detail/kd_17711.html)名称后选择“连接”就可成功访问该无线局域网了。

7、至此，无线局域网设置完成。
