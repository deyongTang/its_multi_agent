3G上网经常掉线案例分析

3G上网经常掉线案例分析以及解决方案。

## 关键词
3G, 网络, 掉线, 分析

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
**现象描述：**

当使用[3G](/detail/kd_17620.html)上[网卡](/detail/kd_17366.html)上网时，会出现掉线的现象，尤其是中国联通3G或者当网速超过240Kb[ps](/detail/kd_17360.html)时会出现此现象

**案例分析：**

此现象由于微软当前 [USB](/detail/kd_17416.html) 调制解调器[驱动](/detail/kd_17488.html)程序 （Usbser.[sys](/detail/kd_17769.html)） 的限制，当试图通过在 Windows 操作系统中使用 Windows Internet E[xp](/detail/kd_17891.html)lorer 从 Internet 下载文件时下载停止响应，然后超时。

 或者当使用具有更快的[数据传输速度](/detail/kd_17560.html)的 USB 调制解调器或等于 240 的千比特 / 秒 （Kbps） 时出现此问题。

**解决方案：**

在微软官方网站通过邮箱自行申请

<http://support.microsoft.com/kb/925681/zh-cn>

或者下载附件中对应系统下的补丁文件，安装即可。
