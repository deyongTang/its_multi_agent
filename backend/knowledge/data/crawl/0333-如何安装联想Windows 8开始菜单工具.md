# 知识库 333

## 标题
如何安装联想Windows 8开始菜单工具

## 问题描述
安装联想Windows 8开始菜单工具的方法。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
密码, 管理, 设置, 显示, 硬件, Windows 8, 查看, 开始菜单

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
**解决方案:**

1、[点击此处](/KnowledgeResources/2013-05-27/101205/联想Win8开始菜单工具.zip) 打开链接后点击“**打开**”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160331/1459424770463101.jpg)

2、双击压缩包中的“**联想Win8开始菜单工具**”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160331/1459424770658372.jpg)

3、若弹出如下提示请点击“**是**”；

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160331/1459424770829960.jpg)

4、安装成功后，点击左下角“**开始**”标志，将打开传统样式开始菜单。

![](https://chinakb.lenovo.com.cn/ueditor/php/upload/image/20160331/1459424770173902.jpg)
