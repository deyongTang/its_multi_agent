# 知识库 236

## 标题
通过系统的Security Center启用或禁用实现控制Windows的安全中心功能

## 问题描述
我们想控制Windows的安全中心功能，打开Security Center服务却无法启动，可以通过修改服务来实现。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作

## 关键词
安全中心, 服务, 启用, 禁用, Security Center, 联想安全中心, windows security

## 元信息
创建时间:2024-12-26|版本:3.0

## 解决方案
1、使用组合键“**Win+R**”调出运行命令框，键入命令“**services.msc**”，点击“**确定**”，如图所示：

![Win10如何关闭Windows安全中心功能？关闭Windows安全中心的方法](https://img.win10d.com/2023/0216/202302161676513177656010.png)

2、若弹出用户帐户控制窗口，点击“**是（Y）**”，

3、找到 “**Security Center 服务**”，双击它打开服务属性，如图所示：

![Win10如何关闭Windows安全中心功能？关闭Windows安全中心的方法](https://img.win10d.com/2023/0216/202302161676513177531776.png)

4、在“**常规**”选项卡中，选择“**启动类型**”为“**自动（延迟启动）**”，然后点击“**服务状态**”下的“**启动**”按钮后，点击“**确定**”按钮，现在请您确认一下此服务是否可以启动？如图所示：

![Win10如何关闭Windows安全中心功能？关闭Windows安全中心的方法](https://img.win10d.com/2023/0216/202302161676513177587623.png)

5、若服务启动仍然失败，请参照上述步骤来启用 “**Remote Procedure Call（RPC）**” 以及“ **Windows Management Instrumentation**” 这两个服务，上述两个服务成功启动之后，请再次尝试启动 “**Security Center 服务**”，检查是否启动成功。
